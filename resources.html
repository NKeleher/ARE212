Resources for R

I’ve compiled some general advice and a lot of resources for R, programming, and general research. Hopefully they will be of some use to you.

Advice on trouble shooting

  • You will learn a lot more if you Google your problems. Please try Google before emailing anyone.
  • Use StackOverflow! You don’t have to directly search StackOverflow, but don’t be intimidated. Some of the threads sound (or are) really complicated, but I often find that the answer is out there if I’m willing to slow down and read.
  • R documentation can be confusing, but it also very helpful—particularly if you can find packages’ vignettes or GitHub README files.
